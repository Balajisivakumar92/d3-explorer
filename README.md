# D3 Explorer

Exploring D3.js (v5): ES2015/ES6, Webpack, D3 and React.js

[Demo Website](https://artyomtrityak.github.io/d3-explorer/)

## How to run
1. `npm install`
2. `npm start`
3. Go to `http://localhost:9000/`
